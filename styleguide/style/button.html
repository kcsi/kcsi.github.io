1.1 _buttons.scss

.buttonのバリエーション。通常のボタン

  • :hover,:active - マウスオーバーと押下時
  • :disabled,.disabled - 押せないとき
  • .-sizeL - 大きいボタン
  • .-sizeS - 小さいボタン
  • .-sizeFull - 横幅いっぱい
  • .-login - ログインボタン
Link button

:disabled,.disabled Link button
.-sizeL Link button
.-sizeS Link button
.-sizeFull Link button
.-login Link button
